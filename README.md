# 易学八卦问事

一个用于每日学习《周易》、练习八卦象义、记录问事解卦的静态网页。

访问地址：

https://allenbj2024.github.io/zhouyi-daily-app/

## 快速使用

1. 在“今日”看一卦，写下应行、应止和复盘标准。
2. 在“八卦”熟悉乾、坤、震、巽、坎、离、艮、兑八个基本象。
3. 在“起卦”写一个具体问题，生成本卦、变卦、动爻和行动建议。
4. 把解读保存到“记录”，三日或七日后回看是否应验在行动上。

不用注册。日课和解卦记录保存在每个人自己的浏览器里。
